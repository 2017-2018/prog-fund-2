Assignment1

Based on materials contained in the first 5 weeks of slide decks and labs. It is worth 50% of your overall grade for the module.

Lab-01

In this lab, you will explore the Eclipse IDE, use the main method as the starting point for your applications and read in various data types from the console using the Scanner class. You will also recap writing encapsulated classes.

Lab-02a

In this lab, you will become more familiar with primitive arrays, particularly using counter-controlled loops with them.

Lab-02b

In this lab, you will become more familiar with the Eclipse IDE, read various data types from the console using the Scanner class and store the details in an array.

Lab-03

On completion of this lab you should be more familiar with the Eclipse IDE, be able to use ArrayLists and have adopted a menu driven approach to the Shop console app.

Lab-04

Implement CRUD into the Shop and DVD projects.

Lab-05

Learn how to store objects into XML files and retrieve them from XML files. We will also implement exception handling for invalid user input.

Lab-06

Build a small, console-based application using ArrayList, HashMap and HashSet, string tokenizing, and the Random class to return random objects from a collection.

ShopV6.0

Packages, Utilities and Parsing: use a package structure for the Shop app and also to write re-usable utilities. We will also experiment with Wrapper classes and parsing.
